# VolBot AI — Торговый ИИ-робот для Deriv Volatility 75

> Автоматическая торговля на синтетическом рынке Volatility 75 через MetaTrader 5. ИИ-модель обучена на тысячах часов исторических данных и работает 24/7 без участия трейдера.

---

## Что в проекте

### Лендинг-сайт (`index.html`)

Одностраничный сайт на чистом HTML/CSS/JS, размещён на GitHub Pages по адресу **vibai.ru**.

**Секции:**
- Hero с превью сделок и графиком
- Счётчики статистики (3 842 сделки, 97% точность, 127 подписчиков)
- «Как работает» — 3 шага подключения
- «О роботе» — панель статуса с live-данными
- Результаты торговли по месяцам (октябрь 2024 — март 2025)
- Для кого подходит (новички / опытные трейдеры)
- Тарифы — подписка $100/мес и курс $397
- FAQ — 6 вопросов с ответами
- Финальный CTA

**Адаптивность:**
- Десктоп (>900px) — двухколоночный layout
- Планшет (600–900px) — одноколоночный layout
- Мобильные (<600px) — полная адаптация шрифтов, кнопок, карточек

**Кнопки CTA** — все ведут на Telegram-бота `t.me/volbotai_bot`.

---

### Telegram-бот (`volbot/`)

Полноценный бот с воронкой продаж, написан на `python-telegram-bot`.

**Для пользователей:**
- `/start` — приветствие и главное меню
- Результаты торговли — история по месяцам
- О роботе — технические детали ИИ
- Как начать — пошаговая инструкция
- Тарифы — карточки подписки и курса
- 7 дней бесплатно — активация триала
- FAQ — 6 вопросов с ответами
- Поддержка — сообщение админу
- `/stats` — быстрая статистика дня

**Для администратора:**
- Уведомление о каждом новом пользователе
- Приём фото чека оплаты + кнопки подтверждения/отклонения
- `/admin` — статистика подписчиков и выручка
- `/broadcast` — рассылка всем пользователям

```

├── index.html              — лендинг-сайт (HTML/CSS/JS)
```

## Деплой

### Лендинг (GitHub Pages)

1. Репозиторий: `github.com/kiosk84/vibai-site`
2. Ветка: `main`
3. Домен: `vibai.ru` (CNAME в репозитории)
4. DNS настроен через Timeweb (4× A-записи + CNAME)

### Telegram-бот

```bash
cd volbot
pip install -r requirements.txt
python bot.py
```

Для постоянной работы — VPS + systemd или screen (см. `volbot/README.md`).
